PluginResettable

public protocol PluginResettable

Note that although this protocol is public, the reset method is intended only for internal use, and should not be invoked by host applications.

  • reset(onComplete:)

    Called when the client calls Amplify.reset(). When invoked, the plugin must release resources and reset shared state. Shortly after calling reset() on the plugin, the category and its associated plugins will be released. Plugins must invoke the onComplete handler when they complete their reset. Immediately after invoking onComplete, the plugin’s underlying system must be ready to instantiate a new plugin and configure it.

    This method is intended only for use by the Amplify system, and should not be invoked by host applications
